# Logic Quest iOS

Versão mobile focada em iPhone para acompanhar a trilha de lógica fora do desktop.

## Como rodar

1. Entre na pasta `ios-app`
2. Instale as dependências com `npm install`
3. Rode `npm run ios` para abrir no simulador

## O que já tem

- Módulos reais carregados da mesma trilha da versão web
- Seleção de lições por módulo
- Tela de leitura e prática da lição atual
- Checkpoint com alternativas clicáveis
- Visual dark pensado para app educacional premium

## Próximos passos naturais

- Persistência local com AsyncStorage
- Navegação com tabs e stack
- Sistema completo de progresso e XP igual ao web
- Publicação via EAS Build
